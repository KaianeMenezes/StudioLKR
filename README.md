# Studio LKR

Landing page desenvolvida para o Studio LKR com foco em presença digital e direcionamento rápido para canais de contato.

## Tecnologias utilizadas
- HTML5
- CSS3
- Font Awesome

## Funcionalidades
- Layout responsivo
- Botões de acesso rápido:
  - WhatsApp
  - TikTok
  - Facebook
  - Endereço no Google Maps

## Objetivo
Criar uma página elegante e objetiva para reunir os principais canais de atendimento e redes sociais do Studio LKR em um só lugar.
